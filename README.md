# Task 6 - Sales Trend Analysis Using Aggregations

## Objective
Analyze monthly revenue and order volume using SQL aggregation functions.

## Tools Used
- SQLite
- DB Browser for SQLite

## Dataset
online_sales table containing:
- order_id
- order_date
- amount
- product_id

## Queries Performed
1. Monthly Revenue Analysis
2. Monthly Order Volume Analysis
3. Top 3 Months by Revenue

## SQL Concepts Used
- GROUP BY
- ORDER BY
- SUM()
- COUNT(DISTINCT)
- Aggregate Functions

## Results
- Monthly revenue and order volume were calculated successfully.
- March 2024 recorded the highest revenue.
- Sales trends were identified using monthly aggregation.
